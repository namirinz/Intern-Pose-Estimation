1.MPHB-label.txt: annotation of the human body in each image in the MPHB dataset

format:

idx:  the index of annotation/image in the MPHB dataset
bbox: the position of the bounding box of human body: <left> <top> <right> <bottom>
source: the original file name of the annotated image in the MPII/LSP dataset

2.train+val.txt: this file gives the index of images used for training and validation.

3.test.txt: this file gives the index of images used for testing.
